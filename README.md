## üîé SQL Techniques Implemented
| Analysis Type | Purpose | SQL Functions Used |
|---------------|----------|--------------------|
| Dimension Exploration | Understand structure & uniqueness | DISTINCT, ORDER BY |
| Date Range Analysis | Identify historical boundaries | MIN(), MAX(), DATEDIFF() |
| Measures Exploration | Compute totals & averages | COUNT(), SUM(), AVG() |
| Magnitude Analysis | Distribution across categories | GROUP BY, ORDER BY |
| Ranking Analysis | Identify top/bottom performers | RANK(), DENSE_RANK(), ROW_NUMBER(), TOP |
| Change Over Time | Track trends & seasonality | DATEPART(), DATETRUNC(), FORMAT() |
| Cumulative Analysis | Running totals & growth | SUM() OVER(), AVG() OVER() |
| Performance Analysis | YoY / MoM comparison | LAG(), CASE |
| Data Segmentation | Business classification logic | CASE, GROUP BY |
| Part-to-Whole Analysis | Contribution comparison | SUM() OVER() |
